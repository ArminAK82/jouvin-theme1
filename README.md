# پوسته ژووین

این یک پوسته سفارشی برای وردپرس است که برای وب‌سایت ژووین طراحی شده است. این پوسته از ووکامرس و المنتور پشتیبانی می‌کند.

## ویژگی‌ها

*   طراحی واکنش‌گرا
*   پشتیبانی از ووکامرس
*   پشتیبانی از المنتور
*   آماده برای ترجمه
*   فرمت‌های مختلف نوشته
*   تصاویر شاخص
*   منوی سفارشی
*   پس‌زمینه سفارشی
*   لوگوی سفارشی

## نصب

1.  پوشه `jouvin-theme` را در مسیر `wp-content/themes/` آپلود کنید.
2.  از طریق منوی "نمایش" > "پوسته‌ها" در پیشخوان وردپرس، پوسته را فعال کنید.

## فایل‌های قالب

این پوسته از ساختار استاندارد فایل‌های قالب وردپرس پیروی می‌کند:

*   `index.php`: قالب پیش‌فرض
*   `front-page.php`: برای صفحه اصلی سایت
*   `single.php`: برای نمایش نوشته‌های تکی
*   `page.php`: برای نمایش برگه‌های تکی
*   `archive.php`: برای نمایش آرشیوها (دسته‌ها، برچسب‌ها و...)
*   `search.php`: برای نمایش نتایج جستجو
*   `404.php`: برای صفحه خطای 404
*   `header.php`: سربرگ سایت
*   `footer.php`: پاورقی سایت
*   `sidebar.php`: نوار کناری
*   `functions.php`: فایل توابع پوسته
*   `style.css`: فایل استایل اصلی
*   `comments.php`: قالب نمایش دیدگاه‌ها

## سفارشی‌سازی

شما می‌توانید این پوسته را از طریق ویرایشگر پوسته در پیشخوان وردپرس یا با ویرایش مستقیم فایل‌ها سفارشی‌سازی کنید.
برای تغییر استایل‌ها، فایل `assets/css/main.css` را ویرایش کنید.
برای افزودن کدهای جاوااسکریپت، فایل `assets/js/main.js` را ویرایش کنید.
